# Design Research Notes

## Prismetric.com Analysis

### Color Palette
- **Primary:** Deep navy blue (#0A0E27 or similar dark blue)
- **Accent:** Purple/magenta gradient for CTAs (#8B5CF6 to #D946EF range)
- **Text:** White on dark backgrounds, dark navy on light sections
- **Backgrounds:** Dark navy for hero, white for content sections
- **Avoid:** This uses purple gradients which user explicitly wants to avoid

### Layout & Spacing
- **Hero Section:** Full-width with large typography, significant padding (80-100px vertical)
- **Content Sections:** Generous whitespace between sections (100-120px)
- **Grid System:** 3-4 column layouts for service cards
- **Container Width:** ~1200-1280px max width
- **Card Spacing:** 24-32px gaps between cards

### Typography
- **Headings:** Large, bold sans-serif (likely Inter or similar)
- **H1:** ~48-56px
- **H2:** ~36-42px
- **Body:** ~16-18px line height 1.6-1.8
- **Font Weight:** Mix of 400, 500, 600, 700

### Trust Indicators
- **Client Logos:** Horizontal scrolling marquee with grayscale logos
- **Stats Counter:** Animated numbers with "+" suffix (Happy Clients, Solutions, Countries, Developers)
- **Testimonials:** Card-based carousel with client name, title, company
- **Case Studies:** Large cards with images and "View Case Study" CTAs

### Key Design Patterns
1. **Hero with Illustration:** Right-side has AI brain illustration with circuit board patterns
2. **Logo Marquee:** Infinite scroll of client logos below hero
3. **Service Grid:** Cards with icons, titles, descriptions
4. **Stats Section:** 4 columns with animated counters
5. **Testimonial Carousel:** Client quotes with navigation dots
6. **CTA Buttons:** Purple gradient, rounded corners, hover effects

### What to Avoid (Generic AI Theme)
- Purple/blue gradients (Prismetric uses this heavily)
- AI brain illustrations with circuit patterns
- Glowing neon effects
- Tech grid backgrounds
- Particle effects

### What to Keep/Adapt
- Clean, professional layout structure
- Generous spacing and whitespace
- Client logo marquee for trust
- Animated stats counters
- Testimonial carousel pattern
- Clear service card grid
- Strong typography hierarchy

## Generic AI Website Clichés to AVOID

### The "Purple Problem"
Research shows AI-generated websites consistently use the same design patterns:

1. **Purple/Blue Gradients**
   - Indigo-500 to purple gradients
   - Blue-to-purple color transitions
   - Neon glows and halos
   - **Solution:** Use solid colors, warm tones, or sophisticated neutrals

2. **Overused Typography**
   - Inter font (most common)
   - Roboto
   - System fonts
   - **Solution:** Consider alternatives like Space Grotesk, Satoshi, or custom fonts

3. **Generic Imagery**
   - 3D floating objects
   - Abstract geometric shapes
   - Brain/neural network illustrations
   - Circuit board patterns
   - Glowing particles
   - **Solution:** Use real photography, custom illustrations, or data visualizations

4. **Layout Patterns**
   - Centered text everywhere
   - Excessive emoji icons
   - Drop shadows on everything
   - Floating cards with blur effects
   - **Solution:** Asymmetric layouts, bold typography, clean cards

5. **Content Clichés**
   - "Diverse team in a meeting" stock photos
   - Generic "AI-powered" messaging
   - Buzzword-heavy copy
   - **Solution:** Specific results, real client photos, concrete metrics
